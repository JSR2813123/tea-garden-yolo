group 權限的集合，可以把一個user放入group，自動就能獲得這個group的權限，可以用來分權限的層級，然後給不同的user，然後可以在/admin裡面新增
/model.py class可以新增一個類別，用於紀錄資料的(給維護用的增讀改刪CRUD)
python manage.py createsuperuser 可以用來新增一個user，要填名稱(登入用到)，密碼(登入用到)，email(免填)
python manage.py makemigrations  -> 用來create model
python manage.py migrate ->用來跑新的model  
python manage.py runserver ->執行本機
.\backend\venv\Scripts\python.exe manage.py runserver 127.0.0.1:8000 ->執行venv本機
